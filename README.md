## 📬使い方  
- ダイス機能
  - 使いたいチャンネル・DMで「/dd」コマンドを使用してください。   
  - オプションを指定せずにコマンドのみで実行すると1d100が振れます。成功/失敗の判定もありません。

###### ！4種類の任意オプションについて！
1. 「回数」オプションは1d100の"1"の部分を表します。ダイスを振る回数を指定でき、指定が無い場合は1回になる設定です。
2. 「面数」オプションは1d100の"100"の部分を表します。ダイスの面数を指定でき、指定が無い場合は100面になる設定です。
3. 「目標値」オプションはCoC（→クトゥルフ神話TRPG）における技能値を表します。これを指定する場合はその値以下が成功、それ以外が失敗の判定となり、CoC第6版に基づいたクリティカル/スペシャル/ファンブルの判定も追加されます。
4. 「シークレット」オプションはシークレットダイス（他人に見えない）を使うためのオプションです。1を指定するとシークレットダイスになります。0を指定するか指定が無い場合は周りにも結果が表示されます。
